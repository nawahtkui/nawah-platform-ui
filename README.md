# Nawah Platform UI

Frontend for the Nawah Web3 platform, focused on the Genesis launch phase.

## Features
- Limited Genesis Drop
- Early-bird counter
- USDT (BEP20) payment flow
- Wallet & Whitelist pages
- Scalable UI architecture

## Purpose
Genesis funds are allocated to:
- Platform subscriptions
- Service activation
- Official domain registrations

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
